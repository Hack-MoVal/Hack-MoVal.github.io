The Event's Programming Language

Python 3 will be the official programming language for the Hack MoVal event. It is a powerful, light and easy to pick up language that will open your doors to many different platforms like Raspberry Pi, Linux, Micro:bit, Internet of Thing devices, and much more.

Getting Python 3 on your computer depends on your operating system. There are many resources on the internet on how to get it. Access the Python official website for the download and installion information.

To learn the basics of the Python 3 programming language, check out the Python 3 documenation. This contains all information on the functions and libraries that you might need for your team project. There will be support from student facilitators.
