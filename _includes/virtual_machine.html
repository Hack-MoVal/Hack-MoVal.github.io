You Will Need VirtualBox

VirtualBox is a powerful tool that will allow you to virtualize a computer inside your computer. You will need it for this event because we will be supplying a pre-configured Ubuntu operating system. It will contain the necessary packages for completing the event.

However, if such a demand cannot be met, we can find alternatives for you. Just make sure to contact us before the event starts and we will help you.

Installing VirtualBox on your computer, just like a programming language, depends on your operating system. Use the VirtualBox official site to download the software executable. The internet is a very useful tool for learning how to install the application.
